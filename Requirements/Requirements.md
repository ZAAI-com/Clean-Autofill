# MailFiller

I need a Chrome extension. When the user clicks on the icon, the Chrome extension fills in the email address of the user.

In the extension settings the users can add their email domain (like mg1.de). 

The extension needs to get the main domain (without subdomains) of the current visited website and add this domain before the @-sign of the email address, which it will fill into a form. For example: The user visits the website subdomain.abc.ai, then the email needs to be "abc.ai@mg1.de".

The extension should fill the current textfield selected with the currors. As second alternative fill in the textfield with the email tag.

Plan each step and build the first version of the Chrom extension.
